# kultra-mega-store-sql
SQL-based data analysis case study on inventory, sales, and shipping performance for Kultra Mega Stores (2009–2012).

## Kultra Mega Stores SQL Capstone Project
A SQL‑driven data analysis case study of inventory, sales, and shipping performance for Kultra Mega Stores (2009–2012), uncovering actionable insights to improve revenue, customer value, and logistics decisions.

---
### Project Summary and Highlight of Skills
**Skills & Tools:** SQL querying (GROUP BY, JOINs, AGGREGATION), data analysis techniques, result interpretation, basic Excel visualization, recommendations based on findings.
